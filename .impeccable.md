## Design Context

### Users
Portfolio and demo reviewers evaluating a Kepler exoplanet analysis dashboard. They need to quickly understand the dataset, inspect exploratory analysis, compare model performance, try live inference, and review interpretable association rules.

### Brand Personality
Credible, cosmic, analytical. The interface should feel like a focused mission-control workspace for astronomical discovery rather than a generic dark dashboard.

### Aesthetic Direction
Use a space-inspired mission-control glass theme: deep astronomical background, restrained glass panels, high-contrast chart surfaces, telescope-console details, and subtle motion. The design should be visually memorable without overwhelming the data.

### Design Principles
- Keep charts and tables readable before adding atmosphere.
- Use glass effects on KPIs and insight panels only where they create useful depth.
- Prefer restrained cosmic color accents over broad neon decoration.
- Make the first viewport feel like an operational science console.
- Support reduced motion and responsive layouts for portfolio review contexts.
